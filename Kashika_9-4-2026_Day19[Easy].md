The Backspace String Compare problem asks you to determine whether two strings are equal after simulating typing in a text editor that supports a special character: #. Each string represents a sequence of keystrokes. Normal characters (a–z) are typed into the editor, while # represents a backspace, meaning the last typed character (if any) is deleted. Your task is to process both strings as if they were typed into empty text editors and then check whether the final results are identical.
If # appears when the string is empty, it does nothing.
You must simulate the effect of backspaces correctly.
The goal is not to modify the original strings but to compare their final forms after processing.
