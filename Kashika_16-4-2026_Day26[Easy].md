This problem solves the classic “invert binary tree” problem using a recursive approach. The structure TreeNode defines each node of the binary tree, containing an integer value and pointers to its left and right children. The main logic lies in the invertTree function inside the Solution class. The function takes the root of a binary tree as input and returns the root of the inverted tree. The first step checks if the current node is nullptr; if so, it simply returns nullptr, which serves as the base case for recursion. If the node exists, the function swaps its left and right child pointers using the swap function. This effectively mirrors the current node. After swapping, the function recursively calls itself on the left and right subtrees (which have just been swapped), ensuring that every subtree in the tree is also inverted. Finally, the function returns the root node after all swaps are completed.
