This problem asks up to swap the nodes in pairs. Basically (1->2->3) should become (2->1->3) as the swapping is done in pairs.
I first tried to use only 2 pointers to the node, one being the first element and the second being the one next to it. It didn't work. But then i tried using dummy node to keep a track on the new head. Again didn't work due to overwriting, losing connection between node etc etc. Then comes the actual correct process. 
I take a dummy node and connect it to the head. The dummy node will be our prev node that will keep track of the new head too. Then I started a loop that works till prev->next and prev->next->next exists. These two condition then take form of actual pointers and with a little bit of rearranging (prev.next = second; first.next=second.next; second.next=first), the first existing pair is swapped. Then we increment prev by 2 (prev=prev.next.next)
