This code generates Pascal’s Triangle up to a given number of rows (`numRows`). It uses a 2D vector `result` to store all rows of the triangle. The outer loop runs from `i = 0` to `numRows - 1`, where each iteration constructs one row. For every row, a vector `row` of size `i + 1` is initialized with all elements set to 1, because the first and last elements of each row are always 1. Then, an inner loop runs from `j = 1` to `i - 1` to fill the intermediate elements. Each element is calculated as the sum of two elements directly above it in the previous row: `result[i-1][j-1]` and `result[i-1][j]`. After computing the row, it is added to `result`. Finally, the function returns the completed triangle. This approach efficiently builds Pascal’s Triangle using previously computed values.
