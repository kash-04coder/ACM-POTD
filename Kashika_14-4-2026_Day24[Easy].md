This problem uses the knowledge of graphs. In this problem, we have to find the centre of the star graph. To do that we use a simple code. Basically we check whether or not there is a common element in the first two of the pair. If the first element of first pair is present in the first element or as the second element of second pair, then that element is the center of the star graph. Otherwise the second element of the pair is the center. It is a simple yet dignified problem with to-the-point syntax.
